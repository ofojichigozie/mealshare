# MealShare Backend

This backend service powers the MealShare application with authentication, household management, meal planning, shopping lists, expenses, and real-time notifications. It is built with NestJS, TypeScript, PostgreSQL, and Drizzle ORM.

## Setup

```bash
git clone https://github.com/ofojichigozie/mealshare.git
cd mealshare/mealshare-backend
cp .env.example .env
npm install
npm run db:push
npm run start:dev
```

The API runs on http://localhost:3000 by default.

For the full project overview and frontend setup instructions, see the root README in the repository.
