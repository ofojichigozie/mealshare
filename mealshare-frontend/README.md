# MealShare Frontend

This frontend application provides the user experience for MealShare. It includes authentication, dashboard views, meal planning, shopping lists, household management, and cost splitting.

## Setup

```bash
git clone https://github.com/ofojichigozie/mealshare.git
cd mealshare/mealshare-frontend
cp .env.example .env
npm install
npm run dev
```

The app runs at http://localhost:5173 by default, assuming the backend is also running on http://localhost:3000.

For the full project overview, environment configuration, and backend instructions, see the root README in the repository.
